The Buyer’s ideal quantity X is chosen as follows:

  1. The computer will draw a ball at random from an urn. The draw will determine the Buyer's ideal quantity.
  2. The values that can be drawn range from 1 to 6. However, not every number is equally likely.
  3. In each round, both the Buyer and the Seller will know which urn the ball is drawn from. There are three types of urns:
    • Low urn, where lower numbers (1s and 2s) are more common
    • Middle urn, where middle numbers (3s and 4s) are more common
    • High urn, where high numbers (5s and 6s) are more common
  4. The Seller then chooses {% if player.single == 0 %} a minimum and maximum quantity {% else %} a quantity {% endif %} to offer to the Buyer.
  5. The Buyer then makes a quantity choice M, either walking away (purchasing M=0 widgets), or choosing to purchase {% if player.single == 0 %} any amount between the seller’s chosen minimum and maximum. {% else %} the seller’s chosen quantity. {% endif %}
  6. Payoffs for the round are determined according to the table for the given quantity choice M and the ideal quantity X.
