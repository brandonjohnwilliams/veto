{{ block title }} Part Four: Overview {{ endblock }} {{ block content }}

Just as in Part Three, you will receive a fixed payment based on the average expected outcome against Robot Buyers. However, your role is now to make Seller decisions for this alternative version of the experiment.

Your Role as the Seller

Your task is to choose {{ if single == 1 }} an offer (a specific number of widgets) {{ else }} a menu (a minimum and maximum number of widgets) {{ endif }} to propose to the Robot Buyers for each of the three urn types (Low, Middle, High).

The Alternative Robot Buyers

The Robot Buyers in Part Four have been trained using the actual decisions of {{ if single == 1 }}64{{ else }}66{{ endif }} participants who completed prior sessions of the alternative version of this experiment.

For any {{ if single == 1 }} offer {{ else }} menu {{ endif }} you propose, the Robot Buyers' response will average across:

  • The different ideal quantities (X) each Robot Buyer draws from the selected urn.
  • The different decisions for how the {{ if single == 1 }}64{{ else }}66{{ endif }} prior participants might respond, {{ if single == 1 }} in taking your offer {{ else }} in choosing an option from your menu {{ endif }}, or walking away.

By taking the average across a large number of Robot Buyers, we remove all randomness.

Your Task

Using your intuition about how Buyers might respond, your task is to choose the {{ if single == 1 }} offer {{ else }} menu {{ endif }} for each type of urn that you believe will generate the highest average expected payment for the Seller in this alternate version of the experiment. Note that the different urns affect how likely each ideal quantity is for the Robot Buyers, and will therefore affect the average payment you receive.

Payment for Part Four

Three different participants in the room will be selected for payment in Part Four, one for each of the three types of urn.

  • Payment Amount: If selected, your payment will be the average expected payment generated by your chosen {{ if single == 1 }} offer {{ else }} menu {{ endif }} against Robot Buyers drawing ideal quantities from the given urn.
