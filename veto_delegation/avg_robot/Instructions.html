{{ block title }} Part Three: Overview {{ endblock }} {{ block content }}

In Part Three, we restore the Buyer's option to walk away and purchase zero widgets. However, you will not interact with human participants in this room. Instead, you will be matched with a large number of Robot Buyers trained to act like those participants.

By matching you to many of these robots and taking averages across their actions, we can eliminate randomness and offer you a fixed payment for any decision. Your role in Part Three is use your experience from Part One to make decisions that you think will maximize this fixed payment.

Your Role as the Seller

You will act as the Seller in Part Three. Just like in Part One, you will make three choices, one for each urn type (Low, Middle, and High).

  • {{ if single == 1 }} You will choose an offer (a specific number of widgets) to propose to the Robot Buyers. {{ else }} You will choose a menu (a minimum and maximum number of widgets) to propose to the Robot Buyers. {{ endif }}

The Robot Buyers

In Part One, there was a degree of randomness in your payoff driven by:

  • the ideal quantity (X) drawn by the Buyer.
  • the specific decisions of the participant you were matched with.

In Part Three, the Robot Buyers you will match with have been trained using the actual decisions of {{ if single == 1 }}66{{ else }}64{{ endif }} participants who completed Part One in prior sessions. For any {{ if single == 1 }} offer {{ else }} menu {{ endif }} you propose, the Robot Buyers' response will average across:

  • The different ideal quantities (X) each Robot Buyer draws from the selected urn.
  • The different decisions for how the {{ if single == 1 }}66{{ else }}64{{ endif }} prior participants might respond, {{ if single == 1 }} in taking your offer {{ else }} in choosing an option from your menu {{ endif }}, or walking away.

Each separate Robot Buyer evaluates a fair draw of an ideal quantity from the given urn against the likely choices of these prior participants. By taking the average across a large number of Robot Buyers, we remove all randomness.

Your Task

Using your experience from Part One, your task is to choose the {{ if single == 1 }} offer {{ else }} menu {{ endif }} for each type of urn that you believe will generate the highest average expected payment for the Seller. Note that the different urns affect how likely each ideal quantity is for the Robot Buyers, and will therefore affect the average payment you receive.

Payment for Part Three

Three different participants in the room will be selected for payment in Part Three, one for each of the three types of urn.

  • Payment Amount: If selected, your payment will be the average expected payment generated by your chosen {{ if single == 1 }} offer {{ else }} menu {{ endif }} against Robot Buyers drawing ideal quantities from the given urn.
