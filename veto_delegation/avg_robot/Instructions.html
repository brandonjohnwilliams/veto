Part Three uses the exact same Buyer-Seller setting as Part One. You will make three separate decisions as the Seller, one for each urn type (Low, Middle, and High) that the Robot Buyers draw their ideal quantities from.

Robot Buyers

In Part One, there was a degree of randomness in your payoff driven by:

  • the specific ideal quantity (X) drawn by the Buyer from the urn.
  • the specific decisions of the participant in the room that you were matched with.

In Part Three, the Robot Buyers have been trained using the actual Buyer decisions of {{ if single == 1 }}66{{ else }}64{{ endif }} participants who completed Part One in prior sessions. For any {{ if single == 1 }} offer {{ else }} menu {{ endif }} you propose, the Robot Buyers' response will average across:

  • The different ideal quantity (X) each Robot Buyer draws from the selected urn.
  • The different decisions for how the {{ if single == 1 }}66{{ else }}64{{ endif }} prior participants might respond, {{ if single == 1 }}taking your offer, {{ else }}choosing an option from your menu, {{ endif }} or walking away.

Each separate Robot Buyer fairly draws an ideal quantity from the given urn and a likely choice from one of the {{ if single == 1 }}66{{ else }}64{{ endif }} prior participants. By taking averages across a very large number of Robot Buyers, we remove all randomness from the Seller's payoff.

Your Task

Using your experience from Part One, your task is to choose the {{ if single == 1 }} offer {{ else }} menu {{ endif }} for each type of urn that you believe will generate the highest average expected payment for the Seller. Note that the different urns affect how likely each ideal quantity is for the Robot Buyers, and will therefore affect the average payment you receive.

Part Three Payment

Three different participants in the room will be selected for payment in Part Three, one for each of the three types of urn.

  • Payment Amount: If selected, your payment will be the average expected Seller payment, calculated by providing your {{ if single == 1 }} offer {{ else }} menu {{ endif }} to many Robot Buyers drawing their ideal quantities from the given urn.
